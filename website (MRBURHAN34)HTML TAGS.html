html:

Defines the root of an HTML document.

head:

Contains metadata about the document, such as title, character encoding, and linked stylesheets.


title:

Sets the title of the HTML document, displayed in the browser's title bar or tab.


meta:

Provides metadata about the document, like character set, author, or keywords for search engines.


link:

Links external resources like stylesheets or icons to the HTML document.


style:

Defines inline CSS styles for the document.


script:

Embeds or references JavaScript code.


body:

Contains the visible content of the web page.


h1, h2, h3, h4, h5, h6:

Create headings with decreasing levels of importance.


p:

Defines a paragraph of text.


a:

Creates hyperlinks to other web pages or resources.


img:

Embeds images.


ul:

Defines an unordered (bulleted) list.


li:

Represents a list item within ul or ol.


div:

A generic container for grouping and styling content.


span:

A generic inline container for styling small portions of text.


table:

Defines a table.


tr:

Represents a table row.


td:

Defines a table cell within a row.


th:

Defines a table header cell.


form:

Creates an input form for user data collection.


input:

Defines various types of form input fields (text, password, radio, etc.).


textarea:

Creates a multiline text input field.


button:

Defines a clickable button within a form.


select:

Creates a dropdown list for selecting options.


option:

Represents an option within a select element.


label:

Labels form elements, improving accessibility and usability.


iframe:

Embeds another web page within the current page.


audio:

Embeds audio content.


video:

Embeds video content.


canvas:

Provides a drawing surface for JavaScript-based graphics.


svg:

Defines Scalable Vector Graphics for vector-based images.


blockquote:

Marks a block of text as a quotation.


cite:

Specifies the title of a work being cited within text.


abbr:

Defines an abbreviation or acronym.


code:

Indicates a snippet of code.


pre:

Preformatted text, preserving whitespace and line breaks.


hr:

Creates a horizontal rule for visual separation.


br:

Inserts a line break within text.


meta:

Specifies character encoding for the document.


base:

Specifies a base URL for relative URLs in the document.
